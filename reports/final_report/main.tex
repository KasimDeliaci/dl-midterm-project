\begin{abstract}
This report studies image classification with feature vectors extracted from ResNet50, MobileNetV2, and EfficientNetB0. The project compares single-backbone representations, pairwise feature fusion, and three-backbone feature fusion under frozen feature extraction and fine-tuning settings. All representations are evaluated with a shared multilayer perceptron classifier using accuracy, precision, recall, and F1-score.
\end{abstract}
